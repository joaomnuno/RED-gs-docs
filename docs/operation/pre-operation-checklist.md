# Pre-Operation Checklist

Actions to complete before powering or connecting the groundstation to the vehicle.

- Hardware inspection (enclosure closed, connectors secure, wiring tags present).
- Power checks (input source, voltage range, DMS test).
- Network checks (link status, addressing).
- Software state (services running, configs correct, time sync).
- Communication check with vehicle/pad.

Keep this in a printable, step-by-step format.
